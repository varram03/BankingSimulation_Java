# Banking System Simulation

The Banking System Simulation is a console-based Java application developed to simulate core banking operations. It allows users to register, log in securely, create bank accounts, and perform transactions such as deposits, withdrawals, and fund transfers. 
The system uses MySQL for persistent data storage and JDBC for database connectivity. This project demonstrates the application of object-oriented programming, database integration, and transaction handling in a real-world banking scenario.

---

## Features
- User Registration and Login Authentication  
- Password Strength Validation  
- Create Savings and Current Accounts  
- Deposit Funds  
- Withdraw Funds  
- Transfer Money Between Accounts  
- View Transaction History  
- Persistent Storage using MySQL Database  
- Exception Handling for Invalid Transactions  
- Commit and Rollback support for secure transactions  

---

## Technologies Used
- **Programming Language:** Java  
- **Database:** MySQL  
- **Connectivity:** JDBC  
- **Concepts Used:**  
  - Object-Oriented Programming (OOP)  
  - Exception Handling  
  - Database Transactions

---

## Project Flow

1. **User Registration/Login**  
   Users register with credentials or log in if already registered.
2. **Account Creation**  
   After login, users can create a savings or current account with an initial balance.
3. **Account Management**  
   Users can view and select their accounts for operations.
4. **Transaction Operations**  
   Users can perform:
   - Deposit  
   - Withdraw  
   - Fund Transfer  
5. **Transaction History**  
   Users can view recent transaction records for a selected account.
6. **Logout**  
   User exits the system securely.
